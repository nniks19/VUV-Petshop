== Description ==
This plugin is used to convert all simple select boxes to select2. Checkout out demo here [Select2 Example](https://select2.github.io/examples.html)

== Installation ==
To install this plugin follow below steps

1. Upload the plugin files to the **'/wp-content/plugins/'** directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the **'Plugins'** screen in WordPress
3. That's it, no other settings needed.

== Frequently Asked Questions ==
= When this plugin is useful? =

This plugin is useful when you need to convert simple select box to select2 so that you can search and select the item.
